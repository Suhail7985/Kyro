# Hackathon Demo Guide (5 minutes)

1. **Landing** — Show premium SaaS homepage, dark mode toggle
2. **Admin** — Login `admin@talentsphere.ai` → company KPIs, hiring funnel, attrition charts
3. **Recruiter** — `recruiter1@talentsphere.ai` → AI leaderboard, bulk resume upload on Jobs page
4. **Careers** — Employee uploads PDF → visual AI analysis (match %, strengths, recommendation)
5. **AI Chat** — Ask "Show top React developers" or use voice mic
6. **Manager** — Approve pending leave from dashboard
7. **Employee** — Clock in/out, view payslip, leave balance

Highlight: **No human intervention** for resume screening pipeline.
